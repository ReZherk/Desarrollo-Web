Function

Que es una Function?

Una funcion es un bloque de codigo,autocontenido, que se puede
definir una vez y ejecutar en cualquier momento. Opcionalmente,una
funcion puede aceptar parametros y devolver un valor.

Las funciones en javaScript son objetos,un tipo especial de
objetos:
Se dice que las funciones son ciudadanos de primera clase porque
pueden asignarse a un valor,y pueden pasarse como argumentps y
usarse como un valor de retorno.

Forma de declarar una funcion 

      function estoEsUnaFuncion() {
        console.log("uno");
        console.log("dos");
        console.log("tres");
      }

Invocar la function 

                estoEsUnaFuncion();

Como retorna una función (return)

        function unaFuncionQueDevuelveValor() {
            console.log("uno");
            console.log("dos");
            console.log("tres");
            // Se añade un comentario para explicar el retorno de la función
            return "La función ha retornado una cadena de texto";
        }
      
        // Para visualizar el retorno de la función, asignamos el resultado a una variable y lo imprimimos en la consola
        let valorDeFuncion = unaFuncionQueDevuelveValor();
        console.log(valorDeFuncion);
    
        
            
Nota:
                

                Si después de un return existen más elementos que se deben ejecutar, estos no lo harán.
                

                Ejemplo:

Como llamar a las partes de un arreglo?

Para realizar esta accion primero pondremos la variable del
arreglo y seguidamente "[]" dentro los [] debe estar la posicion
de la parte que desea llamar.

Como llamar a las subpartes?

Para hacer eso primero debemos ubicarnos dentro de una parte y
luego hacer lo mismo dentro de esa subparte si ubiera mas
subpartes. 

                          console.log(y[3][2]);<=Me devuelve "c"

Objetos

Que es un Objeto?

Imagina un objeto en JavaScript como una caja mágica donde puedes
guardar todo tipo de cosas: desde tu nombre y edad hasta tus
hobbies favoritos e incluso las cosas que puedes hacer, como
saludar a alguien.


*Propiedades: Son como etiquetas dentro de la caja que te ayudan a
organizar tus cosas. Por ejemplo, podrías tener una etiqueta que
dice "nombre" y otra que dice "edad", y en cada una de ellas
guardas la información correspondiente. También puedes tener
etiquetas para cosas más interesantes, como "saludar", que guarda
una función que te permite decir "¡Hola!".




*Métodos: Son como acciones especiales que puedes hacer con las
cosas dentro de la caja. Por ejemplo, si tienes una etiqueta
llamada "saludar" que guarda una función, puedes usarla para decir
"¡Hola!" en cualquier momento.

Tipos de operadores

Que operadores tenemos?

**Operadores aritméticos**: Se utilizan para realizar operaciones matemáticas básicas.

* `+`: Suma dos valores.
* `-`: Resta el segundo valor del primero.
* `*`: Multiplica dos valores.
* `/`: Divide el primer valor por el segundo.
* `%`: Devuelve el resto de una división (operador módulo).

EJEMPLO:

      let l = 5 + (5 - 5) * 3;
      let division = 5 / 2;
      let modulo = 5 % 2;

      console.log(l);<=Devuelve 5
      console.log(division);<=Devuelve 2.5
      console.log(modulo);<=Devuelve 1
Nota:Al igual que en las matematicas los parentesis tiene mayor jerarquía.

Operadores de comparación: Se utilizan para comparar dos valores y devolver un valor booleano (true o false).


  • ==: Compara si dos valores son iguales.
  • !=: Compara si dos valores son diferentes.
  • ===: Compara si dos valores son iguales en valor y tipo (estrictamente iguales).
  • !==: Compara si dos valores son diferentes en valor o tipo (estrictamente diferentes).
  • >: Compara si el primer valor es mayor que el segundo.
  • <: Compara si el primer valor es menor que el segundo.
  • >=: Compara si el primer valor es mayor o igual que el segundo.
  • <=: Compara si el primer valor es menor o igual que el segundo.

Valores unarios


Operador de incremento (`++`):

  • El operador de incremento (`++`) aumenta el valor de una variable en uno.
  • Se puede colocar antes (`++variable`) o después (`variable++`) del nombre de la variable.
  • Si se coloca antes (`++variable`), primero se incrementa el valor de la variable y luego se utiliza en la expresión.
  • Si se coloca después (`variable++`), primero se utiliza el valor actual de la variable en la expresión y luego se incrementa.

EJEMPLO:

let contador = 0;
let resultado1 = ++contador; // Incrementa contador a 1 y luego asigna 1 a resultado1
let resultado2 = contador++; // Asigna el valor actual de contador (1) a resultado2 y luego incrementa contador a 2

Operador de decremento (`--`):

  • El operador de decremento (`--`) disminuye el valor de una variable en uno.
  • Al igual que con el operador de incremento, se puede colocar antes (`--variable`) o después (`variable--`) del nombre de la variable.
  • El comportamiento con respecto a la posición del operador es el mismo que con el operador de incremento.

EJEMPLO:

let contador = 0;
let resultado1 = --contador; // Decrementa contador a -1 y luego asigna -1 a resultado1
let resultado2 = contador--; // Asigna el valor actual de contador (-1) a resultado2 y luego decrementa contador a -2

Operador `+=` (suma y asignación):


  • El operador `+=` se utiliza para incrementar el valor de una variable sumándole otro valor y luego asignando el resultado a la misma variable.
  • La sintaxis es `variable += valor`, donde `variable` es la variable a la que se le asignará el resultado de la suma, y `valor` es el valor que se sumará a la variable.
  • Es equivalente a escribir `variable = variable + valor`.

EJEMPLO:

let numero = 5;

numero += 3; // Equivalente a: numero = numero + 3

// Ahora numero es 8

Operador `-=` (resta y asignación):


  • El operador `-=` se utiliza para decrementar el valor de una variable restando otro valor y luego asignando el resultado a la misma variable.
  • La sintaxis es `variable -= valor`, donde `variable` es la variable a la que se le asignará el resultado de la resta, y `valor` es el valor que se restará de la variable.
  • Es equivalente a escribir `variable = variable - valor`.

EJEMPLO:

let numero = 10;

numero -= 4; // Equivalente a: numero = numero - 4

// Ahora numero es 6

Operadores lógicos: Se utilizan para realizar operaciones lógicas entre valores booleanos.

  • `&&`: Operador AND. Devuelve `true` si ambos operandos son `true`.
  • `||`: Operador OR. Devuelve `true` si al menos uno de los operandos es `true`.
  • `!`: Operador NOT. Devuelve el valor opuesto del operando booleano.
