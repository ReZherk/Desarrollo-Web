Pseudoclases

required

Cuando queremos que el borde de una "input" mantenga un color o cambia a un color cuando esta falte completar ,se usa: 

                            .form-pseudoclases *:required{
                                border-color: yellow;
                            }


Valid

Ahora si queremos que al estar todo correcto en un formulario el borde se vuelva de un color, hacemos esto: 

                            .form-pseudoclases *:valid{
                                border-color: green;
                            }


invalid

Ahora si queremos que los bordes cambien de color a la hora que el formulario fue mal llenado, hacemos esto: 

                            .form-pseudoclases *:invalid{
                                border-color: red;
                            }


focus

Ahora si queremos que los recuadros de los formularios tengan un color a la hora de clikear sobre ellos, hacemos esto: 

                            .form-pseudoclases *:focus{
                                background-color: skyblue;
                            }

Como cambiar color a los enlaces?

Para cambiar el color de los enlaces, tenemos los siguientes:

Enlaces visitados

Para dar un color diferentes a los enlaces visitados del color que viene predeterminado se hace esto. 

                            .form-pseudoclases *:visited{
                                color: olive;
                            }


Color de los enlaces

Para dar un color diferentes a los enlaces, se hace esto: 

                            .menu-pseudoclases a:hover{
                                color: orange;
                            }

Nota:
En el css hay jerarquia y si queremos por ejemplo en el caso de los enlaces poder seguir visitando un enlace vistitado, debemos primer agregar un estilo a visited y despues a hover.

Nota2:

El "hober" no solo se usa en link si no pueden ser aplicadas a cualquier tipo etiqueta.
