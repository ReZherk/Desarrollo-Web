    Lista de etiquetas HTML:

    • <html>: Define el inicio y el final del documento HTML y el elemento raíz del mismo.
    • <meta>: Incluye metadatos en el documento HTML, proporcionando información adicional sobre el mismo.
    • <title>: Define el título del documento HTML, que se muestra en la barra de título del navegador.
    • <link>: Vincula recursos externos al documento HTML, como hojas de estilos, iconos, fuentes, etc.

    Estos son los metadatos que se encuentran en el encabezado del documento HTML:

    • DOCTYPE: Define la versión de HTML utilizada.

    • html lang: Define el idioma del contenido.

    • :charset: Define la codificación de caracteres utilizada.

    • viewport: Define cómo el navegador controla el tamaño y la escala de la página en dispositivos móviles.

    • title: Define el título del documento.

    • description: Proporciona una descripción breve del contenido de la página.

    • stylesheet: Enlaza una hoja de estilo externa.

    • canonical: Especifica la URL canónica del documento.

    • icon: Define el ícono que se mostrará en la pestaña del navegador y en la lista de marcadores.

    • apple-touch-icon: Define el ícono que se usará en dispositivos Apple.

    • theme-color: Define el color del tema del navegador móvil.

    • og:title: Proporciona el título del documento para vistas previas en redes sociales (Open Graph).

    • og:description: Proporciona una descripción del documento para vistas previas en redes sociales (Open Graph).

    • og:image: Especifica la imagen que se utilizará en las vistas previas en redes sociales (Open Graph).

    • og:url: Especifica la URL del documento para vistas previas en redes sociales (Open Graph).

    • twitter:card: Define el tipo de tarjeta de Twitter que se utilizará para la vista previa en Twitter.

    • twitter:site: Especifica el nombre de usuario de Twitter asociado con el sitio.

    • strong: Se utiliza en HTML para dar énfasis al texto.

    • class:La clase en HTML es un atributo que se incluye dentro de la etiqueta de un elemento HTML para identificarlo y aplicar estilos CSS específicos.


    Los tipos de letra

    • Párrafos (<p>): Se utilizan para agrupar bloques de texto o contenido. Son elementos de bloque que crean un nuevo párrafo en el documento.
    • Texto en negrita (<b>): Se utiliza para resaltar el texto haciendo que aparezca en negrita.
    • Texto en cursiva (<i>): Se utiliza para resaltar el texto haciendo que aparezca en cursiva o itálica.
    • Texto subrayado (<u>): Se utiliza para resaltar el texto haciendo que aparezca subrayado, aunque su uso no es común debido a que el subrayado puede asociarse con enlaces en la web.
    • Superíndice (<sup>): Se utiliza para representar texto en una posición superior a la línea de base, comúnmente utilizado para exponentes.
    • Resaltado (<mark>): Se utiliza para resaltar partes de un texto, a menudo con un color de fondo diferente para llamar la atención del lector.
    • Texto en tamaño pequeño (<small>): Se utiliza para reducir el tamaño del texto, generalmente para notas de pie de página o texto menos importante.
    • Subíndice (<sub>): Se utiliza para representar texto en una posición inferior a la línea de base, comúnmente utilizado para subíndices.
    • Texto fuertemente enfatizado (<strong>): Se utiliza para indicar que el texto es especialmente importante o relevante, a menudo se muestra en negrita por defecto.
    • Citas en bloque (<blockquote>): Se utiliza para citar un bloque de texto que debe ser separado del texto principal y con un estilo de presentación especial.
    • Cita del autor (<cite>): Se utiliza para citar el nombre del autor de una cita o referencia.
    • Salto de línea (<br>): Se utiliza para insertar un salto de línea en el texto, generalmente dentro de un párrafo para forzar un cambio de línea.
    • Línea horizontal (<hr>): Se utiliza para crear una línea horizontal que separa el contenido, como secciones diferentes de una página web.
    • Texto preformateado (<pre>): Se utiliza para mostrar texto con un formato fijo, conservando los espacios en blanco y los saltos de línea tal y como están escritos en el código HTML.
    • Código (<code>): Se utiliza para resaltar y mostrar fragmentos de código dentro del texto, a menudo con una fuente de ancho fijo para facilitar la lectura.

    Etiquetas semanticas estructurales

    (Etiquetas de bloque 👆🏻)


    <div>: Es una etiqueta contenedora que no tiene semántica.
    <header>: Cabecera de un sitio web o de una sección.
    <nav>: Representa una navegación.
    <main>: Define la sección principal del documento, solo puede existir una etiqueta main por documento.
    <footer>: Pie de página de un sitio web o de una sección.
    <article>: Representa una sección autocontenida (que por sí sola se explica).
    <aside>: Representa contenido complementario o secundario.
    <section>: Representa una sección de contenido genérico.
    <address>: Representa información de contacto.

    Figuras

    <figure>: Esta etiqueta se utiliza para envolver cualquier contenido independiente, como imágenes, gráficos, ilustraciones, diagramas, código, etc. Ayuda a asociar un título o leyenda con ese contenido.


    <img>: Esta etiqueta se utiliza para insertar una imagen en una página web. El atributo src especifica la ruta de la imagen que se debe mostrar. El atributo alt proporciona un texto alternativo que se mostrará si la imagen no se puede cargar o si el usuario utiliza un lector de pantalla.


    <figcaption>: Esta etiqueta se utiliza para definir la leyenda de una figura (<figure>). Proporciona una descripción o título para la imagen o contenido dentro de la etiqueta <figure>. Es importante para mejorar la accesibilidad web, ya que proporciona contexto adicional sobre el contenido visual para usuarios que no pueden ver la imagen.


    Ejemplo :

    
      
    <figure>
    
        
      <img src="ruta/a/la/imagen.png" alt="tabla de los elementos HTML">
    
        
      <figcaption>Tabla de los elementos HTML</figcaption>
    
      
    </figure>

    2. Elemento 2
    3. Elemento 3

    1. Elemento 3
    2. Elemento 2
    3. Elemento 1

    2. Elemento 2
    3. Elemento 3

    Tablas


    <table>: Esta etiqueta define una tabla en HTML. Es el contenedor principal para todas las filas, columnas y celdas de la tabla.
    <tr>: Esta etiqueta define una fila en la tabla. Dentro de cada <tr> se colocan las celdas de la fila.
    <td>: Esta etiqueta define una celda de datos en una tabla. Especifica el contenido de una celda en una fila.
    <thead>: Esta etiqueta define la sección de encabezado de una tabla. Se utiliza para agrupar las filas que contienen encabezados de columna.
    <tbody>: Esta etiqueta define la sección del cuerpo de una tabla. Contiene las filas que representan los datos principales de la tabla.
    <th>: Esta etiqueta define una celda de encabezado en una tabla. Se utiliza para representar encabezados de fila o columna en una tabla. A menudo se usa en la primera fila o en las primeras columnas de una tabla para identificar los datos en esa fila o columna.
    <th colspan="3">: Esta etiqueta define una celda de encabezado en una tabla y especifica que debe ocupar tres columnas.
    <td rowspan="2">: Esta etiqueta define una celda de datos en una tabla y especifica que debe ocupar dos filas.

    Audio y video


    <audio>
    Este es el elemento HTML utilizado para incorporar archivos de audio en una página web.
    src="media/fondo musical.mp3"
    Este atributo especifica la URL del archivo de audio que se va a reproducir. En este caso, el archivo se llama "fondo musical.mp3" y está ubicado en una carpeta llamada "media".
    controls
    Este atributo agrega controles de reproducción estándar al reproductor de audio, como botones de reproducción, pausa y volumen.
    preload
    Este atributo indica al navegador que cargue el archivo de audio tan pronto como sea posible. El valor preload sin especificar (preload) generalmente significa que el navegador debe cargar el archivo de audio completamente antes de que comience la reproducción.






    Iframes


    La etiqueta <iframe> en HTML se utiliza para incrustar otro documento HTML dentro de la página actual. "iframe" significa "inline frame" (marco en línea) y se utiliza para mostrar contenido de otro origen, como una página web externa, un video de YouTube, un mapa de Google Maps, etc., dentro de la página web actual.

    p>Algunos casos de uso comunes de <iframe> incluyen incrustar contenido de otros sitios web, integrar contenido multimedia como videos o mapas, mostrar contenido de redes sociales, entre otros.








    Como ponerlo en youtube?




    <iframe>: Este es un elemento HTML que se utiliza para incrustar otro documento HTML dentro del documento actual. En este caso, se está utilizando para incrustar el reproductor de video de YouTube en la página.

    width="560" height="315": Estos atributos especifican el ancho y la altura del reproductor de video en píxeles.

    src="https://www.youtube.com/embed/-oK6zL01fNM?si=dqskvrh-w5oZ3IhV": Este es el atributo src que indica la URL del recurso incrustado. En este caso, es la URL del video de YouTube que se va a incrustar.

    title="YouTube video player": Este atributo proporciona un título para el reproductor de video incrustado.

    frameborder="0": Este atributo especifica si se debe mostrar un borde alrededor del reproductor de video. En este caso, se establece en 0 para que no haya ningún borde.

    allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture; web-share": Este atributo especifica qué acciones y características están permitidas en el reproductor de video incrustado, como la reproducción automática, la escritura en el portapapeles, etc.

    allowfullscreen: Este atributo permite que el reproductor de video se expanda a pantalla completa cuando el usuario lo solicita.
